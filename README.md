# Retail-Analytics-Power-BI

Project Overview:

This project focuses on building an end-to-end retail analytics solution using Power BI. The project involves data cleaning, transformation, data modeling, KPI tracking, dashboard creation, and business insights generation to analyze sales, profitability, customer behavior, and regional performance.

The objective of this project is to transform raw retail sales data into meaningful business insights through interactive dashboards and data storytelling.

Problem Statement:

Retail businesses generate large volumes of transactional data, but extracting meaningful insights from this data can be challenging. The goal of this project was to analyze retail sales data to identify:

High-performing product categories
Profitable customer segments
Regional sales and profit trends
Customer purchasing behavior
Areas requiring business improvement

The project also focuses on improving decision-making through interactive dashboards and KPI analysis.

Tools & Technologies Used:
Power BI
Power Query
DAX (Data Analysis Expressions)
Data Modeling
Excel Dataset
Interactive Dashboard Design
Data Cleaning & Transformation

The following data cleaning and transformation tasks were performed:

Handled inconsistent date formats using Power Query
Converted incorrect data types into proper formats
Removed data inconsistencies and errors
Created calculated columns such as Shipping Days
Standardized and validated date columns
Built a Date Table for time intelligence analysis
Data Modeling

A star schema data model was created to improve analytical performance and support dynamic filtering.

Key modeling tasks:

Created relationships between tables
Built a dedicated Date Table
Established one-to-many relationships
Implemented time intelligence structure
DAX Measures Created

The following DAX measures were created:

Total Sales
Total Profit
Total Orders
Profit Margin
Shipping Days Analysis
Dashboard Features

The dashboard includes:

KPI Cards for Sales, Profit, Orders, and Profit Margin
Monthly Sales Trend Analysis
Category-wise Sales Analysis
Regional Profitability Analysis
Customer-wise Sales Analysis
Interactive Slicers for Year, Region, and Segment
Insights & Recommendations Page
Key Insights
Technology category generated the highest revenue across multiple segments.
Consumer segment contributed significantly to total sales.
Certain regions showed high sales but relatively lower profit margins.
A small group of customers contributed a major share of total revenue.
Monthly sales trends indicated possible seasonal demand variations.
Business Recommendations
Increase marketing focus on high-performing Technology products.
Optimize discount strategies to improve profitability.
Strengthen customer retention strategies for high-value customers.
Investigate regions with low profit margins despite strong sales.
Improve strategic targeting for profitable customer segments.
